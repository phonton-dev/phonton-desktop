# Phonton Desktop v0.3.2

## Shell layout

- **Welcome dashboard** when no project is open: recent projects, getting-started steps, sidecar status.
- **Project switcher** in header and sidebar: open folder, recent projects, close project.
- **Sidebar tabs**: Sessions | History with flex scroll lists and status dots.
- **Cleaner header**: sidecar pill + overflow menu (Cloud plans, Settings).

## Workspace UX

- Centered goal composer with example goal chips when idle.
- Focus tabs grouped: Run, Plan, Receipt, Problems + More (Code, Context, Tokens, Log).
- Auto-focus Problems on failure and Receipt on successful completion.
- Right panel renamed to **Handoff** with de-duplicated Receipt copy.

## Requires

**phonton-cli v0.21.0+** for Settings, history, and serve RPC.
