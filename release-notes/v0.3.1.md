# Phonton Desktop v0.3.1

## Bug fix

- Auto-detect stale or outdated `phonton serve` sidecars (< phonton-cli v0.21.0).
- Kill port `:47831` listeners, run `npm install -g phonton-cli@0.21.0+`, and restart sidecar.
- Settings shows an upgrade alert instead of raw `unknown method config.get`.

## UI

- Codex-style hero goal composer with context chips (project, work locally, model).
- Cursor-style session sidebar with search and pin (double-click).
- ADE focus tabs: Plan, Run, Receipt, Problems, Code, Context, Tokens, Log.
- Structured HandoffPacket receipt panel (replaces raw JSON on the right).
- Sidecar status pill in header with upgrade/retry action.

## Requires

**phonton-cli v0.21.0+** for Settings, history, and extensions RPC.
