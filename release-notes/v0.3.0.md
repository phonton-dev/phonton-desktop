# Phonton Desktop v0.3.0 — Agent UI

## UI shell (Cursor / Codex inspired)

- **shadcn/ui** + Tailwind v4 component system
- **Left sidebar**: open project, new goal sessions, run history, doctor shortcut
- **Center**: goal composer + Run / Plan / Output tabs
- **Right**: receipt / review panel (resizable)
- **Settings page** replaces modal — full sidebar navigation

## Projects and sessions

- **Open project folder** (Tauri dialog) — sidecar restarts with project as cwd
- **Multiple goal sessions** per project with session list
- **History** from `tasks.list` RPC — click to resume goal text
- Auto **trust grant** when opening a project

## Settings (in-app)

- Account, appearance, provider, budget, index, permissions, general
- Steering (`steering.toml`) and MCP (`mcp.toml`) editors — global vs project scope
- Inline doctor runner
- App updates

## CLI requirement

Requires **phonton-cli v0.21.0+** for config/tasks/extensions/trust RPC used by settings and history.
